===== Exercise: Fibonacci number sequence =====


The Fibonacci sequence can be generated by starting with the numbers 0,1 and using a simple equation to keep generating new numbers. To generate value $n$, take the value indexed $n-2$ and add it to the value at $n-1$.

!bsubex
Write a program that generates the first 20 numbers in the fibonacci sequence.

!bhint
See if you can make two different programs. One that solves this task using lists, and one that solves it using arrays.
!ehint
